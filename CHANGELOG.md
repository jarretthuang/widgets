# Changelog

All notable changes to this project will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-28

### Added

- Initial stable release for the widgets app.
- Finance stock chart widget builder with symbol presets, theme selection, iframe preview, and direct widget links.
- Countdown widget builder with target-date selection, optional description copy, iframe preview, and direct widget links.
- Light, dark, and system theme support across the launcher and widget configuration pages.
- GitHub Actions release workflow for semver tags.
